2、extend实现

2.1、实现$.extend(a, b)

      var a = { a: 1 }
      var b = { b: 2 }
      var c = $.extend(a, b) 
      得到： c = {
              a: 1,
              b: 2
            }

2.2、实现$.extend(a)/$.fn.extend(a)

      var a = {
        work: function () {
          return '我正在工作'
        } 
      }
      var c = $.fn.extend(a) 
      执行： $().work() 
      输出：我正在工作

2.3、实现$.extend(true, a, b)

      var a = { a: 1, list: {id: 12, name: 'wuzhe'} }
      var b = { b: 2, list: {id: 13, age: 18} }
      var c = $.extend(true, a, b) 
      得到： c = {
        a: 1,
        b: 2,
        list: {
          id: 13,
          name: 'wuzhe',
          age: 18
        }
      }
